Detailed Description

The FeatureExtractor interface is implemented by elements that analyse a time series of Samples to extract a FeatureVector.

Each Pipeline has several FeatureExtractors: one per Signal::Type.

Note
To implement a FeatureExtractor, inherit from BaseFeatureExtractor.
