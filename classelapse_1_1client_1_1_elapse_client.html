Detailed Description

The ElapseClient class provides the main window of the Elapse application.

It contains a Pipeline, a plugin::Manager, and a DeviceProxy which provide all of the actual functionality of the application. It also provides a GUI for interacting with these objects and displaying the state of the signal processing elements.
