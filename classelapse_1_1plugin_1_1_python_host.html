Detailed Description

The PythonHost class loads plugins implemented in Python.

The PythonHost internally launches a Python interpreter which is used to load Python plugins. A plugin is a Python package containing one or more subclasses of elapse::elements decorated with @elapse.export.

Note
Python plugins must be packages, not modules â€” i.e. "foo/__init__.py" not "foo.py".
