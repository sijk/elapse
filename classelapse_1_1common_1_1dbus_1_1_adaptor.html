bool  isFake = false 
)
explicit

Construct a dbus::Adaptor to expose the given device and all of its sub-systems to D-Bus.

This constructor will abort with qFatal() if it fails to connect to the bus and register all objects.

When exposing a simulated device, rather than the real thing, set isFake = true. This will expose the device on the session bus rather than the system bus.
