Detailed Description

The SimpleRawDataSink class dumps raw data to disk using a QDataStream.

The only capture info required for this sink is the directory into which the data files will be saved.

Note that this class only supports saving raw data — not data::Samples or data::FeatureVectors or data::CognitiveState. The saved data can be loaded again by a SimpleRawDataSource.
