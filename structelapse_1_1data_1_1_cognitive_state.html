Detailed Description

The CognitiveState class holds a set of values representing the user's cognitive state.

A Classifier calculates the CognitiveState from a set of FeatureVectors. The number of fields and the meaning of each field in the CognitiveState data structure is left entirely up to each Classifier implementation (which will, of course, document these in detail).
