Detailed Description

The Displayable interface can be implemented by any class that wants to expose a GUI widget.

Member Function Documentation

virtual QWidget* elapse::Displayable::getWidget ( )
pure virtual

Classes implementing the Displayable interface should implement this method to return a widget that depicts the state of the object.

The caller of this function takes ownership of the widget and must delete it at the appropriate time.

Because the lifetime of the widget is managed externally, it would be wise to store the pointer inside this class as a QPointer. This will automatically null the pointer if the widget is destroyed before the object that created it, preventing a dangling pointer.

Implemented in elapse::coreplugin::DummyImuFeatureExtractor, elapse::coreplugin::EegDecoder, elapse::client::BatteryMonitor, elapse::coreplugin::SimpleRawDataSource, and elapse::coreplugin::VideoDecoder.
