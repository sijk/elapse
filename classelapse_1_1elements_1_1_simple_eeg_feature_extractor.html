Detailed Description

The SimpleEegFeatureExtractor class is a convenient base class for implementing EEG FeatureExtractors.

The only method you need to implement is extractFeatures().

protectedpure virtual

Extract features from the given eeg matrix. Each column of the EEG matrix corresponds to a channel of EEG. The number of rows should be equal to the window length times the sampleRate, though it may be slightly less if any samples were dropped. (If you need to detect dropped samples you'll need to inherit from BaseFeatureExtractor instead of this class.)

Returns
a std::vector of extracted features.
