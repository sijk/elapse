Detailed Description

The FeatureVector struct holds a set of meaningful values extracted from a sequence of Samples.

Each FeatureExtractor outputs a FeatureVector for each window of Samples that it analyses. The number of elements and the meaning of each element in the vector is left entirely up to each feature extractor (which will, of course, document these in detail).
