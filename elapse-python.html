elapse¶

All of the python bindings to Elapse are contained within the elapse package. All of the sub-modules are directly accessible from this package, so you don’t need to import them separately — a simple import elapse will suffice.

The purpose of this page is primarily to show which parts of the Elapse API are exposed to Python. For detailed documentation on the classes listed here, follow the links to the documentation of the C++ classes.

class elapse.elements.BaseFeatureExtractor¶

See elapse::elements::BaseFeatureExtractor.

signalType¶

Each subclass of BaseFeatureExtractor must have a class attribute called signalType that indicates what type of signal it works with. The value must be one of the elapse.data.Signal tags.

class MyEegFeatureExtractor(elapse.elements.BaseFeatureExtractor):
    signalType = elapse.data.Signal.EEG
    # ...

elapse.log¶

The functions in this module provide access to the same logger that the C++ code uses. Use these functions instead of the print statement so that messages appear in the application’s log window.

elapse.log.debug('Analysing sample with timestamp', sample.timestamp)

Functions are listed in order of increasing severity.

elapse.settings¶

The functions in this module provide access to the Elapse client application’s settings file. It is strongly recommended that you prefix the names of any settings that you store with 'plugins/myplugin', where myplugin is the name of your plugin, e.g.:

dataDir = elapse.settings.getStr('plugins/mydatasink/datadir')

elapse.gui¶

The functions in this module provide access to miscellaneous GUI dialogs that may be useful for, e.g., implementing DataSink.getCaptureInfo().

elapse.gui.getInt(title, label, value, min, max, step) → int¶

Show a dialog box to request an integer from the user.

Parameters:
  • title (str) – Text to show in the titlebar of the window
  • label (str) – Text describing what should be entered
  • value (int) – Default value in the spin box
  • min (int) – Minimum valid value
  • max (int) – Maximum valid value
  • step (int) – Amount to increment by when pressing the arrow buttons
Returns:

The entered value, or None if cancelled
